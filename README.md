# chariot-rover
Chariot moon rover control system, for ERAU capstone project.

# Libraries 

The Chariot control system utilizes the following libraries: 

    LW20 API (LIDAR) 

    LSM6DSOX (Accelerometer/gyro) 

Components 

    Raspberry Pi Pico 

    Microcontroller board 

    LW20 LIDAR 

    Interface: I2C 

    Address: 0x66 

    LSM6DSOX 

    Interface: I2C 

    Address: 0x6A
